# OPMap

*Exact algorithms to solve the Operator Placement Problem for Data Stream Processing applications*

- - -

OPMap is a command-line based application that allows the user to
solve the Optimal Operator Placement Problem (OPP) for Data Stream Processing
(DSP) Applications.
